## 🚀 Getting Started

### Prerequisites
- Any modern web browser (Chrome, Firefox, Safari, Edge)
- No server-side dependencies required

### Installation
1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/javamastery.git
   cd javamastery
   ```

2. **Open in browser**
   - Simply open `index.html` in your web browser
   - No build process or server required

3. **Start learning**
   - Navigate through the learning path
   - Try the interactive MCQ quiz
   - Explore Java concepts with examples

## 📖 Usage

### Navigation
- Use the sidebar menu for quick navigation between topics
- Responsive navbar with search functionality
- Dark mode toggle for comfortable reading

### Learning Flow
1. **Start with Basics**: Begin with `basic.html` for fundamentals
2. **Explore FAQs**: Check `questionBank.html` for quick concept clarification
3. **Practice with MCQs**: Test knowledge with the quiz system
4. **Explore Advanced Topics**: Continue with advanced concepts
5. **Build Projects**: Apply knowledge with real programs

### Quiz System
- Click "Start Quiz" to begin
- Answer 25 randomly selected questions
- 45 seconds per question
- View detailed score and performance feedback
- Retry anytime to improve

## 📊 Quiz Features

### Question Categories
- **Basic Syntax**: Variables, operators, control flow
- **OOP Concepts**: Classes, objects, inheritance, polymorphism
- **Collections**: Lists, sets, maps, and their operations
- **Multithreading**: Thread creation, synchronization
- **Exception Handling**: Try-catch, custom exceptions
- **Java 8+ Features**: Lambdas, streams, optionals

### Scoring System
- **4 points** per correct answer
- **0 points** for incorrect/unanswered
- **Performance levels**:
  - 90%+: Outstanding (Expert)
  - 80%+: Excellent (Advanced)
  - 70%+: Good (Intermediate)
  - 60%+: Satisfactory (Beginner)
  - <60%: Keep Learning
