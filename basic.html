Java Basics Tutorial

Learn the fundamental concepts of Java programming step by step.

1. Introduction to Java

Java is a high-level, object-oriented programming language developed by Sun Microsystems. It is widely used for building web applications, mobile apps, and enterprise software. 


public class Main {
    public static void main(String[] args) {
        System.out.println("Hello Java!");
    }
}

Interview Questions

Java is an object-oriented programming language used for building cross-platform applications.
Java code runs on JVM which converts bytecode to machine code for any operating system.

2. Java Variables

Variables are used to store data values in Java. 


int age = 20;
String name = "John";

System.out.println(name);

Interview Questions

A variable is a container used to store data values.
Local variables, Instance variables, and Static variables.

3. Java Data Types

Java supports primitive and non-primitive data types. 


int number = 10;
double price = 9.99;
char grade = 'A';
boolean status = true;

Interview Questions

Primitive data types are basic data types like int, char, boolean, double.
The size of int in Java is 4 bytes.

4. Java Loops

Loops are used to repeat a block of code multiple times. 


for(int i=1;i<=5;i++){
   System.out.println(i);
}

Interview Questions

for loop, while loop, and do-while loop.
while checks condition before execution while do-while runs at least once.
